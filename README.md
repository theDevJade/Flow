# Flow

## Kotlin infused playspace for using FlowLang, the custom graph editor, and terminal.
- Custom Flutter app
- WebSocket built
- Performance in mind
- FlowLang - Scripting language made for this
- Graph editor - Flow chart maker that allows execution
- Developer API, modify the language, graph editor, or terminal how you see fit


### The full documentation can be found on https://thedevjade.github.io/Flow/
