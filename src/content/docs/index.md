This is the comprehensive documentation for the Flow project, a powerful development platform consisting of two main components:

## 🚀 FlowLang

**A lightweight, embeddable scripting language for Kotlin/JVM applications**

FlowLang provides a simple syntax for rapid prototyping and automation, with powerful natural language processing capabilities.

[Get Started with FlowLang →](/flowlang/introduction)

### Key Features
- **Lightweight Scripting**: Simple C-style syntax with automatic type conversion
- **Natural Language Processing**: Convert natural language to code
- **Event System**: Built-in event handling and triggering
- **Extensible Architecture**: Custom types and functions
- **Thread-Safe Execution**: Full concurrent access support

## 🎨 Flow Frontend

**A modern Flutter application with dual workspace architecture**

Flow Frontend combines visual graph editing with a full-featured code editor, designed with an Atom-style interface and dark theme.

[Get Started with Flow Frontend →](/frontend/introduction)

### Key Features
- **Dual Workspace**: Visual graph editor and code editor in one app
- **Real-time Communication**: WebSocket-based live updates
- **File System Integration**: Multi-tab editor with syntax highlighting
- **Modern UI/UX**: Dark theme with smooth animations
- **Authentication System**: Mock authentication with session management

## ⚙️ Backend WebServer

**A robust Kotlin-based server providing real-time WebSocket communication and data persistence**

The Flow backend webserver handles authentication, real-time collaboration, file system operations, and graph data management through a comprehensive WebSocket API.

[Explore Backend Documentation →](/backend-webserver)

### Key Features
- **WebSocket Communication**: Real-time bidirectional messaging
- **Authentication System**: Token-based user authentication
- **Database Integration**: SQLite with WAL mode for performance
- **File System Access**: Project file management and operations
- **Graph Synchronization**: Multi-user real-time graph editing
- **Session Management**: Robust connection and session handling

## 🛠️ Quick Start

### For FlowLang
1. Add FlowLang to your Kotlin project
2. Initialize the engine
3. Execute your first script
4. Explore the [Tutorial](/flowlang/tutorial) for more examples

### For Flow Frontend
1. Install Flutter SDK
2. Clone the repository
3. Run `flutter pub get`
4. Start the development server
5. Follow the [Getting Started Guide](/frontend/getting-started)
